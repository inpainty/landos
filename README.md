# Курс фыв

## Перед наяалом фыввы

* убедиться что установлен node.js и npm

> node -v

> npm -v

* убедится что установлен gulp 4

> gulp -v

* Установить 

> npm install gulpjs/gulp-cli#4.0 -g 

## Старт:
* Склонировать
> git clone https://github.com/inpainty/landos.git

* Запустить команду npm install
